# RFL Module 01 GitHub Pages Pack

Upload the contents of this folder to the root of your GitHub Pages repository.

Entry point:
- `index.html`

Core materials:
- `module-01/student-book.html`
- `module-01/workbook.html`

Lesson packs:
- `module-01/lesson-01/` through `module-01/lesson-12/`
- each lesson includes Listening, Drill, Must-Have Inputs, and Audio Map.

Images:
- `module-01/assets/images/`
- `module-01/assets/module01/hero_1920x1080.png`

Audio:
- The HTML files are ready for local MP3 files.
- Browser TTS is not required for the deploy-ready system.
- Add generated audio files into each lesson's `/audio/` folders when available.

GitHub Pages:
- No backend required.
- No database required.
- Static HTML only.
